Setup
  - Our implementation requires pygame and DEAP to be installed

  - Pygame (http://www.pygame.org/wiki/GettingStarted)
    - Download from link OR
    - pip install pygame

  - DEAP (http://deap.readthedocs.io/en/1.0.x/index.html)
    - pip install deap

Running the Game
  - python tetris.py
  - To change weight vector, modify lines 10-13 in tetris.py

Running the Trainer
  - python trainer.py
  - Actual genetic algorithm implemented in main()
